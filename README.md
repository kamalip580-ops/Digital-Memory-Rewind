# Digital Memory Rewind

A Python-based application that stores and retrieves memories digitally.

## Features
- Add memories
- View memories
- Search memories
- User-friendly interface

## Technologies Used
- Python
- Tkinter
- JSON

## How to Run
1. Clone the repository
2. Run:
   python gui.py
